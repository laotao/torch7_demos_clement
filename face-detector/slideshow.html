A simple convolutional network trained end to end to detect faces

  • frames are grabbed from a webcam
  • transformed from RGB -> Y
  • Y is then transformed by the pre-trained network
  • the output of the network is a map of distribution vectors: each vector has 2 components, one for faces and the other one for background
  • this map is then parsed to extract blob locations, which correspond to face locations
